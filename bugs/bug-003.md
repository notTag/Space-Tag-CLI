# [003] Tags on laptop screen only show when that screen is active

- Status: open
- Created: 2026-05-29
- Severity: med
- Bump: patch

## Repro
- Connect to an external monitor.
- Observe the laptop screen's tags.

## Expected
By default, the tags always show.

## Actual
The tags do not appear unless the laptop screen is active.

## Notes
Now that spaces are separated to their respective monitors (reflecting what they
natively have), the default behavior should always show the tags associated with
each screen.

- If there are no CLI commands to set this default, add them.
- If there is no checkbox under Settings in the app, add it.
